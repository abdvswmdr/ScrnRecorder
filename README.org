#+TITLE: ScrnRecorder

ScrnRecorder is a native Qt C++ desktop GUI screen recording application for Linux that ports existing Bash scripts logic (ffmpeg
x11grab).

** Installation

*** Prerequisites

- Linux (X11).
- [[https://www.qt.io/product/qt6][Qt6 (QtCore, QtGui, QtWidgets)]]
- CMake 3.20+
- C++17 (backend)   
- PulseAudio/PipeWire 
- FFmpeg C API (libavcodec, libavformat, libavutil, libswscale, libavdevice).
- GTest for TDD.
